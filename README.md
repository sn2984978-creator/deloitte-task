# Deloitte Task

## Description
This project converts JSON data from two different input formats into a single unified JSON format.

## Files
- `main.py` - Main Python program
- `data-1.json` - Input JSON (Format 1)
- `data-2.json` - Input JSON (Format 2)
- `data-result.json` - Expected output for validation

## Features
- Converts Format 1 to the required output format
- Converts Format 2 to the required output format
- Includes unit tests for validation

## Requirements
- Python 3.x

## Run

```bash
python3 main.py
```

## Output
The program converts both input JSON formats into the expected unified JSON structure and verifies the output using unit tests.
